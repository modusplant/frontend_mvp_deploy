# 폰트 파일 설치 가이드

이 프로젝트는 다음 폰트를 사용합니다:

## Pretendard (본문 폰트)

Pretendard 폰트 파일을 다운로드하여 이 폴더에 추가해주세요.

**다운로드**: https://github.com/orioncactus/pretendard/releases/latest

필요한 파일:
- `Pretendard-Regular.woff2`
- `Pretendard-Medium.woff2`
- `Pretendard-SemiBold.woff2`
- `Pretendard-Bold.woff2`

## Nanum Myeongjo (강조 폰트)

Nanum Myeongjo는 Google Fonts를 통해 자동으로 로드됩니다.
별도의 파일 다운로드가 필요하지 않습니다.

## 임시 대안

폰트 파일이 없어도 프로젝트는 실행됩니다.
fallback 폰트(시스템 기본 폰트)가 사용됩니다.
